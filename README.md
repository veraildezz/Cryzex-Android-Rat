## 📱 Özellikler

- 🎨 **Material 3 UI** - Smooth ve modern tasarımın keyfini çıkarın
- 🤖 **Telegram Entegrasyonu** - Sadece Token ve Telegram ID ile çalışır, tüm loglar botunuza gelir
- 🔒 **Dosya Şifreleme** - Cryzex Rat dosyaları ve kodları şifreler, anlaşılması imkansız hale gelir
- 📦 **Özel Şablonlar** - Kurbanlara ekstra güven verecek özel tasarımlar
